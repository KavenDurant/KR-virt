# 审计模块测试清单

本文档用于验证审计模块的完整功能和集成效果。

- 开发服务器：http://localhost:3003/
- 测试时间：2025年5月26日

### 1. 导航集成测试

- [ ] 左侧活动栏显示审计图标

- [ ] 页面标题显示"审计管理"并带有图标

### 3. 审计日志功能测试

- [ ] 审计日志表格正常加载和显示
- [ ] 表格列包含：时间、用户、操作、资源、结果、风险等级、来源IP、操作

- [ ] 分页功能正常工作

### 4. 安全事件功能测试

- [ ] 安全事件表格正常加载和显示

- [ ] 登录会话表格正常加载和显示

### 6. 配置功能测试

- [ ] 点击"配置"按钮能打开配置模态框
